# 智能排班工具 - 在线版

一个支持自然语言描述排班偏好的智能排班系统，可在线访问使用。

## 🌐 在线访问

部署后可通过以下链接访问：
```
https://你的用户名.github.io/shift-schedule-online
```

## ✨ 功能特性

### 1. 文字描述排班偏好
支持用自然语言描述排班需求：
- **休息日偏好**："每周一固定休息"、"周末尽量不排班"
- **班次偏好**："喜欢早班，不喜欢晚班"
- **特殊日期**："4月5日必须休息"、"五一期间 prefer 上班"
- **同事协调**："希望和张三一起休息"
- **工作强度**："最多连续工作5天"、"希望连休2天"
- **年假使用**："尽量少用年假"

### 2. 班次配置
- 自定义班次名称、时间段、颜色
- 设置最低在班人数要求

### 3. 人员管理
- 创建多组别
- 添加/删除人员

### 4. 假期设置
- 标记节假日
- 设置假期排班规则（优先休息/必须上班/正常排班）

### 5. 智能排班算法
- 自动解析文字偏好
- 避免不合理班次衔接（如晚班后接早班）
- 确保班次均衡分布

### 6. 结果导出
- 可视化排班表格
- 导出 Excel 文件

## 🚀 部署到 GitHub Pages

### 第一步：创建 GitHub 仓库

1. 登录 [GitHub](https://github.com)
2. 点击右上角 **+** → **New repository**
3. 仓库名称填写：`shift-schedule-online`
4. 选择 **Public**（公开）
5. 点击 **Create repository**

### 第二步：上传文件

**方式一：网页上传**
1. 进入刚创建的仓库
2. 点击 **uploading an existing file**
3. 拖拽或选择本项目的所有文件
4. 点击 **Commit changes**

**方式二：Git 命令**
```bash
git clone https://github.com/你的用户名/shift-schedule-online.git
cd shift-schedule-online
# 复制所有文件到该目录
git add .
git commit -m "Initial commit"
git push origin main
```

### 第三步：启用 GitHub Pages

1. 进入仓库的 **Settings**
2. 左侧菜单点击 **Pages**
3. **Source** 选择 **GitHub Actions**
4. 等待 1-2 分钟自动部署

### 第四步：访问网站

部署完成后，访问：
```
https://你的用户名.github.io/shift-schedule-online
```

## 📖 使用说明

1. **配置班次**：设置你的班次信息（名称、时间、颜色）
2. **添加人员**：创建组别并添加人员
3. **描述偏好**：为每个人用文字描述排班偏好
4. **设置假期**：标记特殊日期的排班规则
5. **生成排班表**：点击生成按钮
6. **导出结果**：下载 Excel 文件

## 💾 配置保存与加载

- **保存配置**：点击"保存配置"下载 JSON 文件
- **加载配置**：点击"加载配置"选择之前保存的文件
